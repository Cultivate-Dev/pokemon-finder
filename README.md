# Pokemon Finder

A beginner-friendly web application to search for Pokemon and view their stats, types, and images. Built with vanilla HTML, CSS, and JavaScript using the PokeAPI.

## Quick Start

1. Open `index.html` in your web browser
2. Type a Pokemon name (e.g., "pikachu", "charizard", "bulbasaur")
3. Click Search or press Enter
4. View Pokemon information including stats, type, and sprite

## Technologies Used

- **HTML** - Page structure and layout
- **CSS** - Styling and visual design
- **JavaScript** - API calls and interactivity
- **PokeAPI** - Pokemon data source (https://pokeapi.co/)

## What You'll Learn

- Making API calls with `fetch()`
- Working with JSON data
- DOM manipulation and event handling
- Error handling and user feedback
- Async JavaScript with promises

## Features

- Search Pokemon by name or ID
- Display Pokemon sprite/image
- Show Pokemon type(s)
- View core stats (HP, Attack, Defense, Speed)
- Display physical characteristics (height, weight)
- Error handling with helpful messages
- Enter key support for quick searches
